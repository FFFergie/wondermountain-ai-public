# Wonder Mountain Skills

Put OpenCode skill folders in this directory.

Each skill should be a folder that can be copied directly into:

```text
~/.config/opencode/skills/
```

Guidelines for maintainers:

- Keep one skill per folder.
- Include a `SKILL.md` file in every skill folder.
- Do not depend on `npx`, Node.js, GitHub, or skills.sh for employee installation.
- Do not commit secrets, tokens, private customer data, or credentials.
